Cara Penukaran BRI POIN

Redeem Poin via e-Pay BRI

  • Nasabah belanja di merchant e-Pay (List merchant e-Pay dapat dilihat di aplikasi BRI Mobile-Info BRI-Info Merchant e-Pay)
  • Pilih Metode Pembayaran dengan e-Pay
  • Gunakan poin dengan memasukkan poin untuk mengurangi nilai pembayaran
  • Pesanan segera diproses

Redeem Poin via e-Catalog BRI

  • Nasabah akses ecatalog.bri.co.id
  • Pilih hadiahnya: e-Voucher untuk belanja di merchant, e-Product untuk pulsa dan token listrik, atau barang
  • Masukkan data pesanan: nama, handphone, email, alamat
  • Lakukan pembayaran dengan poin
  • Transaksi berhasil ditandai dengan nasabah terima notifikasi sukses via email
  • E-Voucher dan e-Product maksimal diterbitkan dalam waktu 1x24 jam
  • Untuk e-Voucher dan e-Product akan dikirim via SMS ke handphone nasabah

Redeem Poin via EDC BRI

  • Nasabah berbelanja di merchant kerjasama Bank BRI
  • Pembayaran via EDC BRI:
    1. Menu Purchase
    2. Menu Poin UBB
    3. Menu Redeem Poin
    4. Data Transaksi
    5. Nominal Total Pembayaran
    6. Jumlah Poin yang Akan di Redeem
    7. Pin Kartu Deb
